Module 15: The HTML DOM

The HTML DOM is the interface we use with JavaScript that gives us properties and methods to interact with HTML Elements and CSS Styles. Now that we are familiar with Data Types, Logic, Conditionals, Loops, Functions, and Objects, we are ready to use them to create dynamic web pages! A lot of times programming can seem a little arcane, but when you operate on HTML elements, you will get visual proof of your work in JavaScript. This makes Front-End Development feel much more tangible than Back-End Development. Let's get started!
